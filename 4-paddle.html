This data is taken from a cosmic ray detector test stand in Atlanta, GA. It is located in 1 Park Place, Rm. 706. The graph shows the % variation in cosmic rays detected per hour using a running average. The span of the running average can be adjusted using the slider.

The detectors in this test stand use a scintillating plastic which can detect cosmic ray muon events.
